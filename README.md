把Excel的功能菜单抽象化，然后使用Python模拟这些功能，形成一个可复用的工具模板

工具1：数据体检（迭代中）
打开表格时适用范围较广的一些信息：行数、列数、列名、大概内容；
每一列有多少不同值和不同内容，它们的数据及比例；
有无空行和重复行；
数值列的总和、平均值、中位数、90分位数、计数等

工具2：数据清洗（待实现）
空值填充或者删除
修正标点符号、大小写、空格
重复行是否删除
格式统一，类型修正
是否给表格按某个标准重新排序
写一个排序框架，放哪个列就按哪个列排序
把每个小操作都单独写，但是每次用的时候自己组装，格式处理的，重复行的，空值处理的，统一文本的，用哪个组装哪个
工具二不要和别的工具一样，写成带选项的，比如都是空值，我可以自己选择搭配我是要去删除还是填充

可复跑，可回滚
